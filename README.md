# AIニュース要約通知ツール

## 概要
複数のAI関連ニュースサイトを横断的に巡回し、
記事の要点をAIで要約したうえで、Discordに自動通知する個人用ツールである。

日々の情報収集において発生する
「複数サイトの巡回」「要点整理」といった手作業を自動化することを目的としている。

---

## 主な機能
- AI関連ニュースの自動取得（RSS）
- 記事内容の要約（2〜3行）
- 記事名・引用元・URLを含めた通知
- Discord Webhookによる自動通知
- **指定した時刻に自動実行され、手動操作なしで通知される仕組み**

## 工夫した点
- **Webアプリではなく、Pythonの単体スクリプトとして実装**
  - 個人利用および自動化用途に適したシンプルな構成を採用した
- **本文スクレイピングを行わず、公式に取得可能な情報（RSS）のみを利用**
  - 規約面・安定性を考慮した設計としている
- **生成AI（Gemini）を「作業代替」として活用**
  - 人が行っていた要約・整理作業をAIに任せ、情報収集の効率化を図った
- **通知先をDiscordに設定**
  - 個人でも継続利用しやすく、Webhookを用いることで他ツールへの切り替えも容易な構成としている

---

## 使用技術
- Python
- Gemini API（Google Generative AI）
- Discord Webhook
- RSS

---

## 使い方（概要）
1. スクリプトは指定した時刻に自動実行される
2. 最新のAI関連ニュースを取得
3. Gemini APIを用いて各記事の要点を要約
4. まとめた内容がDiscordに通知される

---

## 想定利用シーン
- AI関連の最新動向を効率的に把握したい場合
- 毎日の情報収集を自動化したい場合
- 要点のみを素早く確認したい場合
